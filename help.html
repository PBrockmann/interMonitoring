Note #1 : Copy a complete monitoring intercomparison tree

You can copy a complete monitoring intercomparison tree produced by the script run on the server by using the following command 

$ mkdir my_intercomparison ; cd my_intercomparison
$ wget -r -A.gif --no-parent -nH --cut-dirs=3 http://webservices.ipsl.jussieu.fr/monitoring/tmp/fegg_plotXXXXXX/images
$ wget -r --no-parent -nH --cut-dirs=3 http://webservices.ipsl.jussieu.fr/monitoring/tmp/fegg_plotXXXXXX/
Note that you download first gif images from the fegg_plotXXXXXX/images directory because the wget command cannot download javascript links.

Note #2 : Save all images and create a pdf file

Proceed as indicated in Note #1 then use the montage command (from ImageMagick) with all downloaded images to produce a pdf document 

$ montage -geometry 100%x100% -tile 2x3 my_intercomparison/images/*gif mydocument.pdf

Note #3 : Run ferret script when you click on a link

You can launch a ferret script (jnl files) directly with ferret run into a xterm window if you specify in your browser that MIME type "application/jnl" with jnl extension has to be opened with the shell script xtermferret.


Note #4 : Prerequesites

All ferret scripts proposed by FEGG use the FAST/ATLAS ferret libraries and need to be run with ferret v608. Please, read http://wiki.ipsl.jussieu.fr/wiki_ipsl/IGCMG/Outils/ferret/Installation00 or install it with following cvs command: 

$ cd /where/to/install/the/package
$ cvs -d :pserver:anonymous@cvs.ipsl.jussieu.fr:/home/ioipsl/CVSROOT login
Type anonymous 
$ cvs -d :pserver:anonymous@cvs.ipsl.jussieu.fr:/home/ioipsl/CVSROOT export -r FAST_608_1_2 fast
$ cvs -d :pserver:anonymous@cvs.ipsl.jussieu.fr:/home/ioipsl/CVSROOT export -r ATLAS_608_1_4 atlas

Note #6 : Automatisation with cron
  1. Download the ferret jnl script and the bash script on you machine in a directory, let's say fegg 
    $ mkdir fegg
  2. Change permission the bash script: fegg_plotXXXXXX_prod.bash 
    $ chmod 755 fegg_plotXXXXXX_prod.bash
  3. Edit the ferret jnl script: fegg_plotXXXXXX.jnl and search pattern "Please modify"
    • Change according to your machine the correct initialisation line for setup FERRET/FAST/ATLAS
      line: .atlas_env_...
    • Put the full pathname to point the ferret jnl script you have downloaded
      line: scriptname=...
    • Copy the resulting directory to somewhere visible from an http server
      line: cp -fR ...

  4. Edit the crontab by typing: 
    crontab -e
    Read more on wikipedia about crontab command

  5. Add entry, here every day at 07h30 
    30 7 * * * /home/yourlogin/fegg/fegg_plotXXXXXX_prod.bash >> /home/yourlogin/fegg/log.txt
  6. Do not forget to remove or coment this entry once the simulations are finished.
